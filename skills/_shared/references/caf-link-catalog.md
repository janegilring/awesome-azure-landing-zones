# CAF/ALZ link catalog

Curated, stable entry-point links for Cloud Adoption Framework (CAF) and Azure Landing Zones
(ALZ), grouped by topic. Skills and agents should link here (or to [`docs/resources.md`](../../../docs/resources.md))
instead of duplicating URLs. Prefer `aka.ms` aliases and `learn.microsoft.com` hub pages — they are stable.

> Source of truth for the full link list is [`docs/resources.md`](../../../docs/resources.md).
> This file organizes the most-used links by design area for quick reuse.
